My Projects (recent to old)

32-bit RISC-V CPU

March - April 2025

Built a CPU that supports 32 bit RISC-V instruction set architecture from scratch in Logisim. Implemented ALU, control logic, immediate generator, 2 stage pipelining, and data hazard solutions.

Voice Recognition

November 2024

Developed a voice recognition system using Singular Value Decomposition (SVD) and Principal Component Analysis (PCA) to classify audio signals based on syllables and intonation using Python. Conducted dimensionality reduction techniques to enhance computational efficiency. Achieved accuracy of average of 97.9% for 4 words.

Ngordnet (N-Grams and WordNet)

October 2024

Efficiently parsed, queried, and analyzed large-scale word frequency data of word usage over default value of 120 years. Constructed a graph model using adjacency list to represent hypernym-hyponym relationships. Implemented BFS-based traversal and recursive helper methods to extract all hyponyms of a given word or set of words.

Shazam

September 2024

Applied Discrete Fourier Transform (DFT) and spectrogram techniques to analyze audio signals and extract meaningful fingerprints using Scipy and Numpy. Implemented peak detection algorithms to identify unique audio features. Achieved accurate song identification using a fingerprint database and hashing techniques.
